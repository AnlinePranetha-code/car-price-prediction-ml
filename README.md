# Car Price Prediction using Machine Learning

This project predicts the price of a car using a Machine Learning model based on
various features such as engine size, horsepower, car width, car length, fuel type,
and aspiration.

## Project Overview
- Type: Supervised Machine Learning
- Algorithm: Linear Regression
- Dataset: Car price dataset (CSV)
- Output: Predicted car price

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Model Evaluation
- R² Score
- Mean Squared Error (MSE)
- Actual vs Predicted Plot
- Residual Plot
- Feature Weight Analysis
